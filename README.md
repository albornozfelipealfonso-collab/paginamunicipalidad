# Sistema de Gestión de Licencias de Conducir

Plataforma web para gestión de citas y emisión de licencias de conducir municipales.

## Características

- Agendamiento de citas en línea
- Validación de RUT chileno
- Restricción de feriados nacionales
- Panel administrativo
- Generación de PDFs oficiales
- Notificaciones por email
- Control de una cita activa por usuario

## Tecnologías

- HTML5, CSS3, JavaScript
- Firebase (Auth, Realtime Database)
- jsPDF
- EmailJS

## Servicios

- Primera Licencia - $25.000 (60 min)
- Renovación - $15.000 (30 min)
- Duplicado - $20.000 (45 min)

## Instalación

1. Clonar repositorio
2. Configurar Firebase en `app.js`
3. Configurar EmailJS
4. Abrir `index.html`
